# Тестирование класса BooksCollector

Этот проект содержит набор unit-тестов для класса `BooksCollector`, который представляет собой систему учета книг с возможностью добавления жанров и избранных книг.

## Покрытие тестами

Тесты покрывают следующие методы и сценарии:

### 1. Метод `add_new_book()`
- ✔ Добавление книги с корректным названием (1-40 символов)
- ✔ Проверка на добавление книги с пустым названием
- ✔ Проверка на добавление книги с названием >40 символов
- ✔ Проверка на добавление дубликата книги

### 2. Метод `set_book_genre()`
- ✔ Установка допустимого жанра для существующей книги
- ✔  Проверка на установку жанра для несуществующей книги
- ✔  Проверка на установку недопустимого жанр

### 3. Метод `get_book_genre()`
- ✔  Получение жанра существующей книги
- ✔  Проверка на получение жанра несуществующей книги

### 4. Метод `get_books_with_specific_genre()`
- ✔ Получение списка книг определённого жанра
- ✔ Получение списка, когда нет книг с указанным жанром

### 5. Метод `get_books_genre()`
- ✔ Получение всего словаря books_genre

### 6. Метод `get_books_for_children()`
- ✔ Получение книг без возрастного рейтинга
- ✔ Проверка исключения книг с возрастным рейтингом

### 7. Метод `add_book_in_favorites()`
- ✔ Добавление существующей книги в избранное
- ✔  Проверка на добавление несуществующей книги
- ✔  Проверка на добавление книги дважды

### 8. Метод `delete_book_from_favorites()`
- ✔ Удаление книги из избранного
- ✔  Проверка на удалиние несуществующей книги

### 9. Метод `get_list_of_favorites_books()`
- ✔ Получение списка избранных книг

## Дополнительные проверки
- Проверка начального состояния жанра у новых книг
- Проверка обработки граничных условий
- Проверка изолированности тестовых случаев
